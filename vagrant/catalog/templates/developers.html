Only for developers

The catalog application website implements a RESTful endpoint. Follow these instructions in order to communicate with this endpoint.

RESTful service for list of items in a category

In order to communicate with the endpoint and retrieve the JSON data for all the items in any given category, first you have to know the id of that pirticular category. Next, you need to go to the following path in order to get the data in JSON Format.


For example if the category id is 0, then the path will be:
{{url_for('wholeCatalogJSON', category_id=0)}} 
      {
        "CatalogName": "Test",
        "CatalogItems": [
          {
            "description": "Test Description 1",
            "id": 0,
            "name": "Item 1"
          },
          {
            "description": "Test Description 2",
            "id": 1,
            "name": "Item 2"
          },
          {
            "description": "Test Description 3",
            "id": 2,
            "name": "Item 3"
          }
        ]
      }

RESTful service for a single item in a category

In order to communicate with the endpoint and retrieve the JSON data for a single item in any given category, first you have to know the id of that pirticular category and then the id of that particular item. Next, you need to go to the following path in order to get the data in JSON Format.

For example if the category id is 0, and the item id is 0 then the path will be:
{{url_for('singleItemJSON', category_id=0, item_id=0)}} 
      {
        "CatalogItem": {
          "description": "Test Description 1",
          "id": 0,
          "name": "Item 1"
        },
        "CatalogName": "Test"
      }
